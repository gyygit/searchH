调用API说明-智能提示

查询条件构造类

com.sinovatech.search.lucenesearch.client.LucenePage

com.sinovatech.search.lucenesearch.client.LuceneQueryUtil

举例说明：

//zhinengword 只能提示的词首字母 例如 北京的北 bj 或者beij 

  String zhinengword = request.getParameter("zhinengword");

//构造Page

 LucenePage page3 =new   LucenePage("sc001",new String[]{"sc0011"}); 

//设置要智能提示的词

page3.setHighLighterWord(zhinengword);

//执行查询

page3 =LuceneQueryUtil.queryIntellisense(page3);

//返回结果

 List<Map<String, String>> data =page3.getListData();
