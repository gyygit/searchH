调用API说明-热词查询

查询条件构造类

com.sinovatech.search.lucenesearch.client.LucenePage

com.sinovatech.search.lucenesearch.client.LuceneQueryUtil

举例说明：

 //查询热词  appcode=接入code                             

            searchWord=null则返回最热词否则是按照        

            searchWord 词进行拼音简拼首字母返回相关热词   

            Count是返回条数                               

LucenePage phot=LuceneQueryUtil.queryHot(appcode, searchWord,count) 

//返回结果

List<Map<String, String>> data =phot.getListData();
